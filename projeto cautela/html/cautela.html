Password Prompt
This script creates a custom prompt class that can be used to display a custom password prompt. The prompt is displayed by calling the prompt() method of the MyPrompt object. The password value can be retrieved by calling the getInputValue() method. The script also allows you to attach event handlers to the submit event of the prompt. Note that this script uses a custom prompt container, rather than the built-in window.prompt() method.
